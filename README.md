## What Is This?

**PC SAVE PROJECT**，中文可意译为 **PC 拯救计划**，是一款使用 Ren'Py 制作的 GALGAME。

标题里的 **SAVE** 同时指向“拯救”和“存档”：在游戏里，真正拯救 PC 的办法，正是依靠你的 save。

This repository is the public release hub: downloads, checksums, release notes, and the project intro page live here.

## Download

Grab the latest build from the release page:

- **Windows/Linux:** `PCSAVEPROJACKT-1.0-pc.zip`
- **macOS:** `PCSAVEPROJACKT-1.0-mac.zip`
